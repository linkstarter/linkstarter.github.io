Markdown语法
2018/03/27 Markdown 2018

希望以后能坚持写博客，第一篇总结一下Markdown的基本语法。

标题

#+空格+标题:

  • 一级标题：# 标题1
  • 二级标题：## 标题2
  • 依次类推

换行

我是第一行（两个空格）
我是第二行

加粗

**我要变粗**

列表

不带序号(减号 + 空格)：

  • item 1
  • item 2

带序号(数字+点+空格)：

  1. item 1
  2. item 2

多级列表（子列表tab缩进）：

  • father-item1
    • child-item1
    • child-item2
  • father-item2

添加图片

感叹号 + 中括号[图片描述性文字] + 小括号(图片相对地址)：

![我是一张图片](images/pic.jpg)

添加超链接

中括号+小括号([something for description](url))：
Mirror Li’s Blog

引用

>大于符号：

真実はいつも一つ(真相只有一个)
–柯南道尔

标亮

波浪号下边的符号(`)

代码块

代码上下行(三个`)

表格

name author
naruto qibenanshi
one piece weitian
